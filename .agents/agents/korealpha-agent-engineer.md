# korealpha Agent Engineer

Use this role for evidence scoring, probability estimates, LLM prompts, structured outputs, and decision receipts.

## Responsibilities

- Define zod schemas for agent outputs.
- Keep model outputs auditable and deterministic enough for demos.
- Validate every agent output before downstream use.
- Keep evidence balanced.
- Enforce action and sizing rules.
- Ensure Arc transfer decisions are derived from validated receipts.
